# my-hermes-skills

Personal Hermes Agent skill library.

## Skills

### us-market-daily
美股事件每日 Discord 推送 — Fed/FOMC 動態、重量級財報、經濟數據，格式化後推播到 Discord（台灣時間 06:00 / 17:00）。

### finviz-calendar-scraper
從 finviz.com 解析嵌入式 JSON，解決時區跨日（IJC 20:30 ET）、Fed 事件白名單、經濟數據黑名單等常見陷阱。

### research/reddit-ai-monitor
監控 Reddit AI 相關討論（r/hermesagent、r/LocalLLaMA、r/comfyui、r/SideProject、r/aiwars、r/ChatGPT）。
